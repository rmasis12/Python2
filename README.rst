==========================================================
Documentacion del sisteman de respaldo de tipos de cambio
==========================================================

Esta aplicacion permite guardar el tipo de cambio de:

* Cambio
* Venta

para usarlo puede ejecutar el siguiente codigo:

.. code-block:: python
    
    import tc as tipo_cambio
    tipo_cambio
